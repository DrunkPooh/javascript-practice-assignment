# JavaScript 실습 과제

## 과제 소개
JavaScript 수업 시간에 진행한 실습 내용을 정리한 과제입니다.  
DOM 선택, 이벤트 처리, 스타일 변경, 조건문을 사용하는 연습을 했고,  
추가로 연산자와 조건문을 활용한 간단한 콘솔 계산기도 만들었습니다.

## 구현한 내용
- 버튼 클릭 시 콘솔에 문장 출력
- 버튼 클릭 시 텍스트 변경
- input 입력값 실시간 출력
- 마우스 올렸을 때 글자색 변경
- 버튼 클릭 시 글자 색상과 크기 변경
- 점수 입력 후 합격 / 불합격 판별
- 콘솔 계산기 기능 구현

## 사용한 언어
- HTML
- CSS
- JavaScript

## 파일 설명
- `index.html` : 화면 구성
- `script.js` : JavaScript 기능 구현
- `과제_제출용_코드.txt` : 제출용 코드 정리 파일

## 실행 방법
1. 파일을 다운로드합니다.
2. `index.html` 파일을 브라우저에서 실행합니다.
3. 필요한 기능을 직접 클릭해보면서 확인합니다.
4. 콘솔 확인이 필요한 기능은 개발자도구(F12)에서 확인할 수 있습니다.

## 느낀 점
이번 과제를 하면서 이벤트 연결하는 방법과 조건문 사용하는 방법을 다시 연습할 수 있었습니다.  
특히 버튼 클릭이나 입력값 감지처럼 화면에서 바로 반응이 보이는 부분이 재미있었습니다.  
아직 JavaScript가 익숙하진 않지만, 직접 만들어보면서 조금씩 이해가 되는 것 같습니다.
