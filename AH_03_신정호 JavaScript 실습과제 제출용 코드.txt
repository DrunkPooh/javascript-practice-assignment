[JavaScript 실습 과제 제출용 코드]

1. index.html 파일과 script.js 파일을 함께 실행하면 됩니다.

2. 구현한 기능
- 버튼 클릭 이벤트
- 텍스트 변경
- 실시간 입력 감지
- 마우스 감지
- 스타일 변경
- 점수 판별
- 콘솔 계산기

3. 실행 방법
- index.html 파일을 브라우저에서 실행
- script.js는 JavaScript 기능 담당
- 개발자도구(F12) 콘솔에서 결과 확인 가능

4. 콘솔 계산기 설명
- 버튼 클릭 시 prompt 창이 뜸
- 숫자 2개와 연산자 입력
- 연산 결과가 콘솔에 출력됨
- 잘못된 연산자 입력 시 안내 문구 출력
- 0으로 나누는 경우 오류 문구 출력
